# 🚀 Hướng Dẫn Deploy Landing Page - Sửa Điện Thoại Khang Huy

## ✨ 3 Cách Deploy Nhanh & Miễn Phí

---

## **CÁCH 1: Netlify Drop (NHANH NHẤT - 30 giây)** ⭐⭐⭐

### Bước 1: Vào Netlify Drop
- Link: https://app.netlify.com/drop

### Bước 2: Kéo thả file
- Kéo file `landing-page.html` vào trang
- Chờ upload xong (2-3 giây)

### Bước 3: Nhận link public
- Netlify tự động tạo link: `https://xxxxx.netlify.app`
- Copy link này để chia sẻ

### ✅ Ưu điểm:
- Không cần tài khoản
- Không cần cấu hình
- Tên miền miễn phí
- Deploy ngay lập tức

---

## **CÁCH 2: GitHub Pages (Miễn phí, Chuyên nghiệp)** ⭐⭐⭐

### Bước 1: Tạo GitHub account
- Link: https://github.com/signup
- Đăng ký email + mật khẩu

### Bước 2: Tạo repo mới
- Click "New repository"
- Tên: `khanghuy-repair` (hoặc tên khác)
- Chọn "Public"
- Click "Create"

### Bước 3: Upload file
- Click "Add file" → "Upload files"
- Chọn file `landing-page.html`
- Commit changes

### Bước 4: Kích hoạt GitHub Pages
- Settings → Pages
- Source: main branch
- Chờ 1-2 phút

### Bước 4: Nhận link
- Link: `https://[username].github.io/khanghuy-repair/`

### ✅ Ưu điểm:
- Tên miền cố định
- Có thể custom domain
- Version control (lịch sử thay đổi)
- Chuyên nghiệp

---

## **CÁCH 3: Vercel (Cách Tối Ưu)** ⭐⭐⭐⭐

### Bước 1: Vào Vercel
- Link: https://vercel.com

### Bước 2: Connect GitHub
- Click "Continue with GitHub"
- Đăng nhập GitHub

### Bước 3: Import repo
- Click "Import Project"
- Chọn repo bạn vừa tạo
- Deploy

### Bước 4: Nhận link
- Vercel tự động cấp link: `https://khanghuy-repair.vercel.app`

### ✅ Ưu điểm:
- Deploy tự động (mỗi khi push)
- Tên miền chuyên nghiệp
- Analytics xem lượt truy cập
- Nhanh nhất

---

## **CÁCH 4: Dùng Custom Domain (Tên Miền Của Riêng Bạn)**

Sau khi deploy ở Netlify/Vercel/GitHub Pages:

### Bước 1: Mua domain
- Các trang: Godaddy, Namecheap, Tên miền việt, VNH
- Giá: ~50K-150K/năm

### Bước 2: Trỏ domain vào Netlify/Vercel
- Vào trang quản lý domain
- Thêm DNS records theo hướng dẫn
- Chờ 12-24 giờ

### Bước 3: Xong!
- Truy cập: `https://suadienthoaikhanghuy.vn`
